## 1. 概述

在天地图离线瓦片地图基础上，实现网格热力展示、网点标注、行政边界定位、数据筛选排名等数据可视化功能，为网点布局优化提供直观的决策支持。

### 1.1 技术栈约束

| 项目 | 说明 |
|---|---|
| 地图引擎 | Leaflet 1.7.1 |
| 地图底图 | 天地图离线瓦片（EPSG:3857），通过 `TiandituBd09Crs` 实现 BD09 坐标透明转换 |
| 坐标体系 | 用户数据统一 BD09，瓦片 WGS84，CRS 层自动转换 |
| 前端框架 | Vue 2 + Element UI |
| 页面路径 | `/jwmap/tianditu` |
| 后端 | Spring Boot 2.5 / MyBatis |

### 1.2 页面布局

```
┌──────────────────────────────────────────────────────┐
│ [顶部工具栏]  行政区 ▼  🔥 热力图开关        │
├──────────────┬───────────────────────────────────────┤
│              │                                       │
│  [侧边栏]    │                                       │
│              │              地图区域                  │
│  正常 380px   │                                       │
│  展开 600px   │       [📍标点] [📏测距]              │
│              │       [边界选择器]                     │
│  浮动覆盖     │                                       │
│  不压缩地图   │                                       │
└──────────────┴───────────────────────────────────────┘
```

- 侧边栏为浮动覆盖层，不压缩地图区域
- 顶部工具栏整合行政区筛选、热力图开关
- 年份选择器位于侧边栏内，在展示网点信息或网点排名时使用

## 2. 功能需求

### 2.1 需求 1：行政边界展示与快速定位

**数据源**：`public/data/map_data/` 目录下的 GeoJSON 文件（BD09 坐标）  
**数据文件**：将 `E:/coding/wangdianxuanzhi/map/src/data/map_data_bd09/` 下的 10 个 `*_full.json` 文件拷贝至前端 `public/data/map_data/`

**功能**：
- 页面加载后默认显示贵州省边界 + 9 个地市边界（不同颜色区分）
- 顶部工具栏行政区下拉框：省→地市→区县三级联动
  - 贵州省 → 定位到全省视图（zoom 10）
  - 选择地市 → 高亮该地市边界，显示区县边界（zoom 11）
  - 选择区县 → 高亮该区县边界（zoom 13）
- 全部使用现有 `BoundaryManager` 逻辑复用

---

### 2.2 需求 2：网格热力展示

**触发流程**：
1. 用户先选择地市（顶部工具栏或边界下拉框）
2. 点击左上角的 **"🔥 热力图"** 按钮 → 显示当前地市的网格热力
3. 再次点击该按钮 → 隐藏热力图
4. 切换地市时热力图自动清除，需重新点击展示

**数据接口**：
- `/jwmap/data/grid/list?city=xxx` — 获取网格元数据（包围盒 + 得分）
- `/jwmap/data/grid/score/byCity/{city}` — [**新增**] 获取该城市所有网格得分列表

**渲染方式（方案 A+B 结合）**：
- **底层**：Canvas 热力叠加层 — 以网格中心点为圆心，得分为权重，渲染连续热力渐变面，实现颜色插值过渡
- **上层**：透明 `L.rectangle` 点击热区 — 与热力层位置对齐，用于捕获点击事件

**配色方案**（得分 0~1，低 → 高）：
```
低分(0) → [绿] → [黄] → [红] ← 高分(1)
         #00ff00  #ffff00  #ff0000
```

**过渡效果**：
- **颜色插值**：相邻网格不做阶梯式硬边界，用 Canvas 热力层实现颜色模糊过渡
- **缩放过渡**：地图缩放时网格跟随平滑缩放，不闪烁

### 2.3 需求 3&4：侧边栏设计与网点展示

**触发方式**：
- 点击网格（透明矩形热区）→ 展示网格信息
- 点击网点图标 → 展示网点信息

**展示形式**：左侧滑出面板（浮动覆盖，不压缩地图）

**面板状态**：

| 状态 | 宽度 | 适用场景 |
|---|---|---|
| 正常模式 | 380px | 仅网格信息 / 仅网点信息 |
| 展开模式 | 600px | **网格内有网点时**自动切换 |

**年份选择器**：位于侧边栏内部，网点信息区域和排名列表使用

**网点标注图标**：
- 自定义 DivIcon：红色圆形底 + 白色"工"字，大小约 24×24px

### 2.4 需求 5：按行政区和支行管辖维度筛选

**顶部工具栏布局**：

```
┌──────────────────────────────────────────────────┐
│  行政区: [贵州省 ▼]  一级支行: [全部 ▼]         │
│                          [🔥 热力图]             │
└──────────────────────────────────────────────────┘
```

**筛选逻辑**：
- 行政区筛选 → 地图定位 + 过滤该区域内的网点
- 一级支行筛选 → 仅显示该支行管辖的网点
- 年份选择 → 位于侧边栏内，影响网点得分数据的查询年份

**排名列表**：

- 筛选后在地图右下角或侧边栏底部展示 **得分排名列表**
- 默认展示 **前 20 名**
- 列表项：`排名 | 名称 | 得分`
- 底部有 **"查看更多"** 按钮（分页加载）
- 点击列表项 → 地图 `flyTo` 定位到对应元素 + 侧边栏更新

**排名列表示例**：


## 3. 交互流程汇总

```
页面加载
  │
  ├─→ 显示贵州省边界 + 各地市边界
  ├─→ 顶部工具栏：行政区=贵州省
  ├─→ 侧边栏：隐藏
  │
  ├─→ 用户选择地市
  │    ├─→ 地图 flyTo 到该地市
  │    ├─→ 高亮地市边界，显示区县边界
  │    └─→ [热力图] 按钮可点击
  │
  ├─→ 用户点击 [🔥 热力图]
  │    ├─→ 加载当前地市网格数据
  │    ├─→ Canvas 热力层 + 透明矩形叠加
  │    └─→ 按钮变为 [隐藏热力图]
  │
  ├─→ 用户点击网格（无网点）
  │    ├─→ 左侧滑出侧边栏(380px)
  │    ├─→ 展示综合得分 + 排名
  │    └─→ 展示分项指标(人口/商业/企业)
  │
  ├─→ 用户点击网格（有网点）
  │    ├─→ 侧边栏展开至 600px 对比模式
  │    ├─→ 左栏：网格资源（得分/指标/位置）
  │    └─→ 右栏：网点经营（得分/分项/基础信息）
  │
  ├─→ 用户点击网点图标
  │    ├─→ 侧边栏(380px)切换为网点内容
  │    ├─→ 展示综合得分 + 5类分项得分(含进度条)
  │    └─→ 展示网点基础信息
  │
  ├─→ 用户点击 [📋 查看详细指标数据表]
  │    └─→ 弹窗展示网格/网点详细指标数值表
  │
  ├─→ 用户选择行政区/支行筛选
  │    ├─→ 地图过滤
  │    └─→ 排名列表更新
  │
  └─→ 用户点击排名列表项
       └─→ 地图 flyTo + 侧边栏更新
```

## 4. 后端 API 变更清单

| # | 接口 | 方法 | 说明 | 状态 |
|---|---|---|---|---|
| 1 | `/jwmap/data/grid/score/byCity/{city}` | GET | 返回城市所有网格得分列表 | ❌ 新增 |
| 2 | `/jwmap/data/grid/indicators/{gridCode}` | GET | 返回单个网格的全部分项指标值 | ❌ 新增 |
| 3 | `/jwmap/data/branch/score/detail/{branchId}/{year}` | GET | 返回单个网点全部分类得分(5类) | ❌ 新增 |
| 4 | `/jwmap/data/grid/ranking/{city}` | GET | 返回网格全市排名(分页) | ❌ 新增 |
| 5 | `/jwmap/data/branch/ranking/{city}/{year}` | GET | 返回网点全市排名(分页) | ❌ 新增 |
| 6 | `/jwmap/data/grid/branches/{gridCode}` | GET | 返回网格内的网点列表 | ❌ 新增 |

**已有但可复用**：
- `/jwmap/data/grid/list?city=xxx` — 网格元数据（含得分）
- `/jwmap/data/branch/list?city=xxx` — 网点列表
- `/jwmap/data/branch/score/{city}/{year}` — 网点得分（当前仅 overall）
