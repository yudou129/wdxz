# 地图数据展示功能 — 开发顺序

## 前提
后端 Maven 已 install、前端 npm 依赖已装、GeoJSON 边界数据需先拷贝到 `ruoyi-ui/public/data/map_data/`

---

### Phase 1：后端新增 API

`JwDataController.java` 新增 6 个接口，新增/修改 3 个 Mapper：
1. `GET /grid/score/byCity/{city}` — 网格得分列表
2. `GET /grid/indicators/{gridCode}` — 网格分项指标
3. `GET /branch/score/detail/{branchId}/{year}` — 网点 5 类分项得分
4. `GET /grid/ranking/{city}` — 网格排名分页
5. `GET /branch/ranking/{city}/{year}` — 网点排名分页
6. `GET /grid/branches/{gridCode}` — 网格内网点列表

`data.js` 同步新增 6 个前端调用函数。

---

### Phase 2：顶部工具栏 + 网点标注

- **TopToolbar.vue**（新建）：行政区下拉框、一级支行下拉框、热力图开关按钮
- **网点 DivIcon**：红底白"工" 24×24px，加载 branchList 生成 Marker 图层，支持按支行筛选

---

### Phase 3：热力图

- **heatmapLayer.js**（新建）：Canvas 热力叠加层（绿→黄→红渐变）+ 透明矩形点击热区
- 集成到 tianditu.vue：加载/隐藏/城市切换清除

---

### Phase 4：侧边栏

- **SidebarPanel.vue**（新建）：浮动左侧栏，支持 380px/600px
- 三种模式：纯网格信息 / 纯网点信息（含 5 类得分进度条）/ 网格+网点对比（600px 双栏）
- 年份选择器（仅网点模式下显示）
- "查看详情"弹窗展示完整指标数据

---

### Phase 5：排名列表 + 筛选联动

- **RankingList.vue**（新建）：前 20 名 + 分页查看更多，点击 flyTo 定位
- 筛选联动：行政区/支行 → 过滤网点 + 排名更新

---

### 依赖关系

```
Phase 1（后端 API）
  ├──→ Phase 2（工具栏 + 网点标注）──→ Phase 5（排名 + 筛选联动）
  └──→ Phase 3（热力图）──→ Phase 4（侧边栏）──┘
```

Phase 2 和 Phase 3 可并行。

---

### 涉及文件

**后端**：`JwDataController.java` + `JwGridDataRawMapper`/`JwBranchScoreMapper`/`JwBranchInfoMapper`（各 +1 查询）

**前端**：`tianditu.vue`（主页面集成）、`data.js`（+6 API）、3 个新组件（`TopToolbar`/`SidebarPanel`/`RankingList`）、1 个新工具（`heatmapLayer.js`）
